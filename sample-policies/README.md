# Sample Policies

This folder contains sample insurance policy documents for testing the InsurAI system.

## Included Samples

1. **Health Insurance Policy** - A comprehensive health insurance policy with medical coverage details
2. **Property Insurance Policy** - Commercial property insurance with coverage terms and conditions
3. **Business Liability Policy** - General liability insurance for business operations

## Usage

These sample policies can be used to:

- Test the upload functionality
- Verify AI analysis capabilities
- Demonstrate the chatbot features
- Validate policy comparison tools

## File Formats

All sample policies are provided in PDF format for consistency. You can also use DOCX files for testing.

## Adding New Samples

To add new sample policies:

1. Place PDF or DOCX files in this directory
2. Update this README with descriptions
3. Ensure files are under 10MB in size
4. Test with the upload functionality

## Note

These are sample documents for testing purposes only and do not represent actual insurance policies or legal documents.
